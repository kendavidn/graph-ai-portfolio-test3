About The Course

"Generative AI for Work & Research" is an 8-week intensive cohort course designed to teach responsible use of AI tools for task automation, document generation, and data analysis without writing complex code.

01

Websites & Docs

Understanding LLMs, generating structured Markdown/HTML, and building simple research websites.

02

Automated Analysis

Using Julius AI and Code Interpreter to process datasets, visualize results, and create reports.

03

Text Analysis

Literature reviews with Elicit, sentiment analysis, and knowledge synthesis from text collections.

04

Custom Chatbots

Building RAG systems, custom GPTs, and interactive chatbots for knowledge management.

05

Task Automation

Designing AI agents for email automation, app-to-app workflows, and browser control.

06

Communication

Creating AI-generated video abstracts, scientific figures, and prompt engineering for visuals.
